  ```bash

  ```

  ```env
  DATABASE_URL=your-database-connection-string
  ```
4. Run database migration:
  ```env
   npx prisma migrate dev --name init
  ```
5. Insert initial data (optional):
  ```env
   npx prisma db seed
  ```
